üéØ Harmonic Color Calculation

Generates complementary, analogous, triadic, and wave-based color palettes using mathematical relationships derived from harmonic frequencies. Every color is calculated, never hardcoded.

üåä Modulation Engine

Four sophisticated modulation techniques: Phase (PM), Frequency (FM), Amplitude (AM), and Combined modes. Watch colors animate with sine, triangle, sawtooth, or square wave patterns.

üéµ Audio-Reactive Visualization

Maps audio frequency bands directly to color properties: bass to hue, mids to saturation, treble to lightness. Capture browser tab audio or microphone input.

üé™ Color Organ Mode

Full-screen immersive color visualization synchronized with audio. Each frequency band displays as a dynamic color block that responds to your music or input.

üí´ Cascade Effect

üìä Real-Time Adjustment

üé® Live Preview Swatches

See primary, accent, secondary, and neutral colors updating in real-time. Hover over any swatch in the harmony cards to see the exact hue value.

üíæ Export Palettes

üîÑ 9 Harmony Modes

üí° Wave Theory Integration

Unlike traditional RGB or HSL color pickers, WB derives colors from mathematical wave relationships. Bass frequencies modulate hue using phase modulation, mids control saturation through amplitude, and treble shapes lightness via frequency shifts. This creates a unified color system grounded in physics.

üî¨ Technical Foundation

The system uses Web Audio API for real-time frequency analysis, CSS custom properties for dynamic theming, and requestAnimationFrame for smooth 60fps animations. Modulation is calculated using mathematical wave functions (sine, triangle sawtooth, square) applied to HSL color space. Audio analysis splits the frequency spectrum into bass (0-15%), mids (15-50%), and treble (50-100%) bands.
