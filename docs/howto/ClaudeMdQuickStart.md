### When You Edit a claude.md File:

## ğŸš¨ Troubleshooting

### "File not being aggregated"
- Check first line has correct marker syntax
- Verify marker has no extra spaces: `<!-- STATUS: UPDATED -->`
- Run `npm run check-status` to verify marker is detected

### "All files show as MISSING"
- Run first-time setup: `npm run init-markers`

### "currentstatus.md seems wrong"
- Tell AI: "Force full claude.md refresh"

### "Script won't run"
- Check PowerShell execution policy
- Run as admin if needed
- Verify scripts folder exists
