Programmatic Control

Click buttons above to test API methods

Code Examples

// Initialize viewport simulator
WBViewport.init({
    showControls: true,
    defaultViewport: 'auto'
});

// Set specific viewport
WBViewport.setViewport('mobile');
WBViewport.setViewport('tablet');
WBViewport.setViewport('desktop');

// Get current viewport
const current = WBViewport.getViewport();

// Reset to auto
WBViewport.reset();

// Add custom viewport
WBViewport.addCustomViewport('custom', 600, {
    name: 'Custom 600px',
    icon: 'üìê',
    color: '#8b5cf6'
});

Event Listeners

// Listen for viewport changes
document.addEventListener('wbViewportChange', (event) => {
    console.log('Viewport changed:', event.detail);
    // event.detail = { viewport: 'mobile', width: 400 }
});

// Listen for component ready
document.addEventListener('wbViewportReady', (event) => {
    console.log('Viewport component ready');
});

üí° Integration Tips

  • Add <script src="../wb-viewport/wb-viewport.js"></script> to any demo page
  • The viewport controls automatically appear at the top
  • Use CSS classes like .wb-viewport--mobile for responsive styles
  • The component respects existing responsive CSS media queries
  • Works seamlessly with all WB components including modals
