üöÄ WB Modal Component

A comprehensive modal dialog component with multiple sizes, variants, and built-in dialog types. Features accessibility compliance, keyboard navigation, and customizable animations.

üöÄ Features

  • ‚úÖ Multiple Sizes: small, standard, large, fullscreen
  • ‚úÖ Theme Variants: default, success, warning, danger
  • ‚úÖ Built-in Dialogs: alert, confirm, prompt with Promise-based API
  • ‚úÖ Animation Options: fade, slide, scale transitions
  • ‚úÖ Modal Stacking: Support for multiple modals with z-index management
  • ‚úÖ Accessibility: ARIA compliant with proper focus management
  • ‚úÖ Keyboard Navigation: Escape to close, tab navigation, focus trap
  • ‚úÖ Responsive Design: Adapts to all screen sizes
  • ‚úÖ Event-Driven: Custom events for show/hide with integration hooks

üíª Basic Usage

<!-- Direct Web Component Usage -->
<wb-modal id="example-modal" title="Modal Title" size="standard" variant="default">
    <p>Modal content goes here</p>
    <div slot="footer">
        <button onclick="document.getElementById('example-modal').removeAttribute('open')">Close</button>
    </div>
</wb-modal>

üéØ JavaScript API

// Create modal programmatically
const modal = WBModal.create({
    title: 'My Modal',
    content: '<p>Content here</p>',
    size: 'large',
    variant: 'success',
    animation: 'fadeIn',
    buttons: [{
        text: 'Close',
        variant: 'primary',
        onclick: () => WBModal.hide(modal)
    }]
});

// Show modal
WBModal.show(modal, triggerElement);

// Built-in dialogs (Promise-based)
const result = await WBModal.alert('Message', 'Title');
const confirmed = await WBModal.confirm('Are you sure?', 'Confirm');
const input = await WBModal.prompt('Enter value:', 'default', 'Prompt');

üì° Event Handling

// Listen for modal events
document.addEventListener('wbModalShow', (e) => {
    console.log('Modal showing:', e.detail);
});

document.addEventListener('wbModalHide', (e) => {
    console.log('Modal hiding:', e.detail);
});

document.addEventListener('wbModalReady', (e) => {
    console.log('Modal component ready');
});

‚öôÔ∏è Configuration Options

  • size: "small" | "standard" | "large" | "fullscreen"
  • variant: "default" | "success" | "warning" | "danger"
  • animation: "fadeIn" | "slideDown" | "scaleUp"
  • title: Modal header title (optional)
  • buttons: Array of button configurations with variants and callbacks
