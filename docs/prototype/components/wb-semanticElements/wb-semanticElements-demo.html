wb-semanticElements Demo

Semantic Elements Auto-Generation

This page demonstrates the automatic creation of semantic HTML5 elements. The component will check for missing elements and create them with visual indicators.

Test Area: Existing Content

This content exists in the page. Notice how the wb-semanticElements component has automatically generated any missing semantic elements and marked them with visual indicators.

What You Should See:

  • Purple dashed borders around auto-generated elements
  • Labels like "<nav> (auto)" for generated elements
  • Semi-transparent backgrounds on auto-generated sections
  • Proper DOM order: nav → header → main → footer

Implementation Notes:

  • Only missing elements are created - existing ones are left untouched
  • Elements receive data-wb-semantic="true" attribute
  • Visual indicators help identify auto-generated structure
  • Works with any existing page content

DOM Structure Check

Open browser DevTools to inspect the DOM structure and see:

  1. Which semantic elements were auto-generated
  2. The data-wb-semantic="true" attributes
  3. Proper element nesting and order
  4. CSS styling applied to generated elements
