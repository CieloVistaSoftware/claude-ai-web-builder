WB Styles System

All components use the centralized styles system:

  • main.css - Entry point (imports all foundation styles)
  • _variables.css - CSS custom properties and design tokens
  • _base.css - Reset and foundational styles
  • _utilities.css - Utility classes
  • layouts.css - Layout system

Component Structure

Each component has its own folder with:

  • wb-component.js - Web Component code
  • wb-component.css - Component-specific styles
  • wb-component.md - Documentation
  • wb-component.schema.json - Schema definition
