üë§ Avatar Placeholders

Small square placeholders for user avatars with initials. The first five use different shades of purple.

HTML

                    
JavaScript

                    
// Dynamically generate avatar placeholders
const avatarData = [
  { label: 'AA', color: '#a78bfa' },
  { label: 'BB', color: '#8b5cf6' },
  { label: 'CC', color: '#7c3aed' },
  { label: 'DD', color: '#6d28d9' },
  { label: 'EE', color: '#4c1d95' },
  { label: 'FG', color: '#22c55e' },
  { label: 'HI', color: '#f59e0b' },
  { label: 'JK', color: '#ef4444' },
  { label: 'LM', color: '#0ea5e9' },
  { label: 'NO', color: '#ec4899' }
];
const avatarDemo = document.getElementById('avatar-demo');
avatarData.forEach(({ label, color }) => {
  const el = document.createElement('wb-placeholder');
  el.setAttribute('width', '60');
  el.setAttribute('height', '60');
  el.setAttribute('color', color);
  el.setAttribute('label', label);
  avatarDemo.appendChild(el);
});

üîß JavaScript API

            
Control the placeholder programmatically.

            

                

                    
                    

                        Default
                        Square
                        Wide
                        Portrait
                        üé≤ Random
                    

                

                

                    
JavaScript

                    
// Get the component
const placeholder = document.getElementById('apiDemo');

// Set individual properties
placeholder.width = 400;
placeholder.height = 200;
placeholder.color = '#6366f1';
placeholder.label = 'Default';

// Or use setOptions() to set multiple at once
placeholder.setOptions({
    width: 300,
    height: 300,
    color: '#22c55e',
    label: 'Square'
});
