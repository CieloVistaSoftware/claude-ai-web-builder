# WB Hero Component

A hero section for landing pages and feature highlights.

## Features
- Large headline and subtitle
- Optional background image
- Call-to-action button
- Responsive layout

## Usage
```html
<wb-hero headline="Welcome to WB!" subtitle="Build beautiful web apps faster." cta="Get Started" bg="hero.jpg"></wb-hero>
```

## Attributes
- `headline`: Main headline text
- `subtitle`: Supporting subtitle text
- `cta`: Call-to-action button text
- `bg`: Background image URL

## Styling
Use CSS variables for colors and spacing.
