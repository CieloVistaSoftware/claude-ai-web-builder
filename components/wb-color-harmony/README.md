# WB Color Harmony Component

## Overview
`wb-color-harmony` is a standalone web component for visualizing a color harmony palette. It displays a set of color swatches with labels, suitable for use with palettes generated by `wb-color-bar`, `wb-color-bars`, or any harmonic color logic.

## Features
- Visualizes an array of color objects (with `hue`, `saturation`, `lightness`, and `name`)
- Simple, flexible, and embeddable in any page or component
- Works with palettes from `wb:color-harmony-change` events

## Usage
```html
<wb-color-harmony></wb-color-harmony>
<script src="wb-color-harmony.js"></script>
```

## Setting the Palette
```js
const harmony = document.querySelector('wb-color-harmony');
harmony.palette = [
  { name: 'Primary', hue: 240, saturation: 70, lightness: 50 },
  { name: 'Complement', hue: 60, saturation: 70, lightness: 50 },
  // ...
];
```

## Integration Example
Listen for a harmony event and update the component:
```js
document.addEventListener('wb:color-harmony-change', e => {
  document.querySelector('wb-color-harmony').palette = e.detail.textPalette;
});
```
