This demo shows a minimal custom element extending WBBaseComponent. Check the event log for lifecycle messages.

JavaScript Usage

import { WBBaseComponent } from './wb-base/wb-base.js';

class MyComponent extends WBBaseComponent {
    constructor() {
        super();
        this.shadowRoot.innerHTML = `
             

                        WBBaseComponent is working!
                    

        `;
    }
    connectedCallback() {
        this.logInfo('MyComponent connected');
    }
}
customElements.define('my-component', MyComponent);

Click the button above to fire a custom event from the wb-demo-base component. The event will be logged in the event log.

JavaScript for Custom Event

// Fire a custom event from the component
const demoBase = document.querySelector('wb-demo-base');
