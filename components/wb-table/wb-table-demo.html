A flexible data table component with JSON data binding, custom column formatters, sorting, and dynamic row management.

HTML
<wb-table 
    data-json='[
        {"id":1,"name":"John Doe","email":"john@example.com","role":"Admin"},
        {"id":2,"name":"Jane Smith","email":"jane@example.com","role":"User"}
    ]'>
</wb-table>

üéØ Basic Usage

Load data from JSON, add rows dynamically, and clear the table.

0 rows
JavaScript
// Get table element
const table = document.getElementById('users-table');

// Set columns
table.setColumns([
    { key: 'id', title: 'ID', width: '10%' },
    { key: 'name', title: 'Name', width: '30%' },
    { key: 'email', title: 'Email', width: '40%' },
    { key: 'role', title: 'Role', width: '20%' }
]);

// Load data
table.setData([
    { id: 1, name: 'John Doe', email: 'john@example.com', role: 'Admin' }
]);

// Add row
table.addRow({ id: 2, name: 'Jane', email: 'jane@example.com', role: 'User' });
