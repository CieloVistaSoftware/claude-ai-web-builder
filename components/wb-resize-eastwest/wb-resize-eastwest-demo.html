üìê How It Works

‚úÖ Hover over the right edge of the sidebar

You'll see the resize cursor (‚ÜîÔ∏è). Drag left or right to resize the sidebar.

  • side="right" - Handle is on the right edge
  • min-width="200" - Sidebar can't be smaller than 200px
  • max-width="600" - Sidebar can't be larger than 600px
  • storage-key - Width persists to localStorage

üéØ Usage

Add the component inside the element you want to resize:

<div id="sidebar" style="position: relative; width: 300px;">
    <wb-resize-eastwest 
        storage-key="sidebar-width"
        min-width="200"
        max-width="600"
        side="right"
        handle-width="8px">
    </wb-resize-eastwest>
    
    <div class="sidebar-content">
        <!-- Your content -->
    </div>
</div>

‚öôÔ∏è Attributes

  • storage-key - localStorage key to persist width
  • min-width - Minimum width in pixels (default: 200)
  • max-width - Maximum width in pixels (default: 1200)
  • handle-width - Width of drag area (default: 8px)
  • side - Handle position: "left" or "right" (default: "right")
  • target - CSS selector of element to resize (default: parent)

üì° Events

  • wb:resize-start - Fired when dragging starts
  • wb:resizing - Fired continuously while dragging
  • wb:resize-end - Fired when dragging ends

üí° Common Use Cases

  • Navigation Sidebars - Left or right side navigation panels
  • Inspector Panels - DevTools-style inspector panes
  • File Browsers - File tree explorers
  • Chat Panels - Messaging or notification sidebars
  • Settings Panels - Configuration or preferences panes

üé® Try It!

  • Drag the sidebar edge to see wb:resizing events
  • Release to see the wb:resize-end event with final width
  • Refresh the page - your sidebar width is saved!
