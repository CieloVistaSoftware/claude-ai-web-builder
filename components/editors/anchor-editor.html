Code Usage

<!-- Include the universal editor script --> <script src="universal-element-editor.js"></script> <!-- Make any element editable by adding data-editable="true" --> <a href="https://example.com" data-editable="true">Editable Link</a> <button data-editable="true" data-style="primary">Editable Button</button> <img src="image.jpg" alt="Description" data-editable="true"> <p data-editable="true">Editable text</p>

JavaScript API

// Make existing elements editable ElementEditor.makeEditable('a, button, img, p'); // Get element state const state = ElementEditor.getState(element); // Set element state ElementEditor.setState(element, { text: 'New text', style: 'button' }); // Get element type const type = ElementEditor.getType(element);
