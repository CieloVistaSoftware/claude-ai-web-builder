Anchor Customization

Anchors and links can be customized with different properties. The anchor editor allows you to:

  • Change link text and URLs
  • Set link targets (same window, new window)
  • Configure link types (internal, external, email, phone)
  • Add anchor IDs for page navigation
  • Set link relationships and attributes
