# wb-linkedinAd

A WB2 custom element for displaying LinkedIn-style ads.

## Properties
- `ad-id`: Unique identifier for the ad
- `company`: Name of the company
- `headline`: Ad headline text
- `cta`: Call-to-action button text
- `dark`: Enables dark mode styling

## Usage
```html
<wb-linkedinAd ad-id="12345" company="Acme Corp" headline="Boost your B2B reach!" cta="Get Started" dark></wb-linkedinAd>
```
