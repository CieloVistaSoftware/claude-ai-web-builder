üîß JavaScript API

Control buttons programmatically.

Click Me Toggle Disabled Cycle Variant
JavaScript
// Get button reference
const btn = document.getElementById('jsDemo');

// Read properties
console.log(btn.variant);  // 'primary'
console.log(btn.disabled); // false

// Set properties
btn.variant = 'success';
btn.disabled = true;
btn.size = 'large';

// Listen for events
btn.addEventListener('wb-button:click', (e) => {
    console.log('Button clicked!', e.detail);
});
