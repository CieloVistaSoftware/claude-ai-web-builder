You can customize which CSS variables the control panel updates:

<wb-control-panel-v2 css-primary="--my-color"></wb-control-panel-v2>

Attributes available:

  • css-primary - Main color variable (default: --primary)
  • css-hue - Hue variable (default: --text-primary)
  • css-sat - Saturation variable (default: --text-primary)
  • css-light - Lightness variable (default: --text-primary)
