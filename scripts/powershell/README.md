# scripts/powershell Directory

This directory contains PowerShell scripts for the WB framework.

## Contents
- `build-and-run-socket.ps1` - PowerShell script to build and run WebSocket test
- `create-favicon.ps1` - PowerShell script to create favicon
- `reorganize-project.ps1` - PowerShell script to reorganize project
- `run-all-wb-tests.ps1` - PowerShell script to run all WB tests
- `server/start-api.ps1` - PowerShell script to start API server
- `test-unified-server.ps1` - PowerShell script to test unified server

## Purpose
PowerShell automation scripts for building, testing, and managing the WB framework.
