# scripts/utils Directory

This directory contains utility scripts for the WB framework.

## Contents
- `convert-to-base-unit-test.js` - Test script for base unit conversion
- `final-validation-test.js` - Final validation test script
- `fix-conversion-syntax-errors.js` - Script to fix conversion syntax errors
- `fix-import-paths.js` - Script to fix import paths
- `quick-test.js` - Quick test script
- `reorganize-project.ps1` - PowerShell script to reorganize project
- `run-all-wb-tests.bat` - Batch script to run all WB tests
- `run-all-wb-tests.ps1` - PowerShell script to run all WB tests

## Purpose
Utility scripts for testing, validation, fixing issues, and project management.
