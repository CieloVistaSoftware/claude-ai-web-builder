# run-and-save.ps1

**Purpose:**
Runs `npm install` and `npm test`, saving their outputs to files for review.

**Usage:**
```
pwsh ./run-and-save.ps1
```

**Output:**
- `install-output.txt` (npm install output)
- `test-output-2.txt` (npm test output)

**Notes:**
- Useful for quickly setting up and testing the project, with logs for troubleshooting.
