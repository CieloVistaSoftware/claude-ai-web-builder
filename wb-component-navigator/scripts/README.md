# scripts/README.md

This folder contains utility scripts for WB Component Navigator.

- `delete-old-md.ps1`: Deletes all `.md` files in the parent folder except those in `docs/`.
- Other setup, install, or diagnostic scripts can be placed here for organization.

## Usage

To run a script, open PowerShell or Command Prompt in this folder and execute the desired script file.

Example:
```
pwsh ./delete-old-md.ps1
```
