# Changelog

All notable changes to AI Verifier will be documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-11-15

### Added
- Initial release of AI Verifier platform
- TypeScript backend (1,273 lines)
- .NET backend (1,241 lines)
- React frontend (608 lines)
- Three verification algorithms:
  * Consistency Checker
  * Ensemble Validator
  * Fact Checker
- Four verification levels (Quick, Standard, Thorough, Critical)
- Complete documentation (8 files, 25,000 words)
- Business plan with market analysis
- Docker support
- Azure deployment guides
- Example tests and scripts

### Technical Stack
- Node.js 18+, Express.js, TypeScript
- .NET 8.0, ASP.NET Core
- React 18, TypeScript
- Anthropic, OpenAI, Brave APIs
