# src/entry Directory

This directory contains the main entry points for the WB framework application.

## Contents
- `index.html` - Main HTML entry point
- `index.js` - Main JavaScript entry point
- `index.css` - Main CSS entry point
- `wb.html` - WB-specific HTML entry point
- `wb.js` - WB-specific JavaScript entry point
- `wb.css` - WB-specific CSS entry point

## Purpose
Entry points for different ways to access and run the WB framework.
