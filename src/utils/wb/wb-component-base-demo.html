This demo shows how to use the WBComponentBase class as a foundation for custom web components in the WB system. See the claude.md file for full details and standards.

  • All components should extend WBComponentBase for shared logic.
  • Follow the CSS-first and two-tab demo standards.
  • See components/claude.md for project-wide conventions.
