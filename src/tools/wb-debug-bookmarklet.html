üéØ Usage Methods

1. Query Parameter

Add ?debug=true to any WB component demo URL:

http://localhost/wb-button-demo.html?debug=true http://localhost/wb-select-demo.html?wb-debug=true

2. Browser Console

Open browser console (F12) and run:

// Enable debug mode WB.debug.enable() // Toggle debug mode WB.debug.toggle() // Get debug stats WB.debug.getStats() // Export debug data WB.debug.exportDebugData()

3. Keyboard Shortcut

Press Ctrl+Shift+D on any page that has the debug manager loaded.

4. Manual Script Injection

Include the debug manager in any page:

<script src="../../shared/wb-debug-manager.js"></script>
