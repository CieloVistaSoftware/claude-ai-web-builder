Quick Start

The system is already installed and working! No additional setup required.

# Run component discovery node tools/component-discovery.js # Build with validation node tools/build-symbols.js build # Watch mode for development node tools/build-symbols.js watch

Integration with Package.json

Add these scripts to your package.json:

{ "scripts": { "discover": "node tools/component-discovery.js", "build:symbols": "node tools/build-symbols.js build", "validate:components": "node tools/build-symbols.js validate", "dev:watch": "node tools/build-symbols.js watch", "prebuild": "npm run build:symbols" } }
