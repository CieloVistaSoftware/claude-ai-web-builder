# BaseUnitTestSimple.js

This file provides a simplified base class for Playwright and unit tests in the Website Builder project. It is intended for use in tests that do not require the full complexity of `BaseUnitTest.js`.

- **Purpose:** Offers lightweight setup and utility methods for simple test cases.
- **Usage:** Extend or instantiate in your test files for basic setup and assertions.
- **Key Features:**
  - Minimal beforeEach/afterEach hooks
  - Basic error monitoring
  - Simple DOM assertion helpers
