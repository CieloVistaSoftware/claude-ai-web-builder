# WBFunctionalTestHelpers.js

This file contains functional test helper utilities for the Website Builder project. It provides reusable functions for advanced Playwright and integration tests.

- **Purpose:** Supplies helper functions for complex test scenarios, such as simulating user interactions and validating event flows.
- **Usage:** Import and use these helpers in your Playwright or integration test files.
- **Key Features:**
  - User interaction simulation (clicks, drags, keyboard)
  - Event and state validation utilities
  - Advanced DOM manipulation for test setup
