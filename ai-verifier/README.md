# AI Verifier

This folder contains the core implementation and documentation for the AI Verifier platform as described in the project changelog.

## Features
- TypeScript backend
- .NET backend
- React frontend
- Three verification algorithms:
  - Consistency Checker
  - Ensemble Validator
  - Fact Checker
- Four verification levels: Quick, Standard, Thorough, Critical
- Example tests and scripts
- Docker and Azure deployment guides

## Technical Stack
- Node.js 18+, Express.js, TypeScript
- .NET 8.0, ASP.NET Core
- React 18, TypeScript
- Anthropic, OpenAI, Brave APIs
